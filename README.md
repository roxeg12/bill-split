Bill Split is an app that makes splitting the dinner bill with a large party easier!




Current features:
-
- Receipt image to text! No need to spend 5 minutes just punching numbers, simply take a picture of the receipt and the items, quantities, and prices will automatically populate.
  -  OpenCV
  -  Tesseract
  -  OpenAI API
    
Features in development:
- 
- Bill API
  - consists of items, their quantities, and their prices, with an overall total that can be split in different ways
  - can create a bill manually or modify a generated one
- Uneven bill splits:
  - split by percentage, by product, and more!
 
Features for the future:
-
- Collaborative bill building
  - can add others on the app to the bill, and pay through the app
